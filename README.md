# Bahia Bus — install

**Install page:** https://abdellah-didou.github.io/bahia-bus/install.html
**Web app:** https://abdellah-didou.github.io/bahia-bus/

Scan to install (routes to the right option per device):

<img src="install_qr.png" alt="Scan to install Bahia Bus" width="240">

- **Android** — download the APK (native app): https://github.com/abdellah-didou/bahia-bus/releases/latest/download/bahia-bus.apk (allow "install from unknown sources").
- **iPhone / iPad** — open the web app, then Share → *Sur l'écran d'accueil* (installs a full-screen PWA — the only free option Apple allows without a paid developer account).

Planificateur de trajets bus & tram pour Oran — itinéraires, lignes, arrêts proches, guidage en direct (FR / AR). Works offline after first open.

This repo holds the built web app + the Android APK release. Source code is in a separate private repository.
